# Sports Statistics Data Analysis

## 📌 Project Description
This project analyzes sports performance data using Python. It helps in identifying trends, comparing team performance, and finding top performers.

## 🛠 Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## 📊 Features
- Data cleaning and preprocessing
- Team-wise performance analysis
- Visualization using graphs
- Identification of top performers

## 📁 Files
- project.py → Main Python code
- sports_data.csv → Dataset

## ▶️ How to Run
1. Install libraries:
   pip install pandas numpy matplotlib seaborn
2. Run:
   python project.py

## 📈 Output
- Bar Chart (Team Performance)
- Histogram (Performance Distribution)
- Line Graph (Yearly Trend)
